Developers

Project Overview

This website was developed as an educational web development project that combines cultural storytelling with modern digital design. The main purpose of this project is to showcase the history, significance, and cultural beauty of the Eyo Festival in Lagos, Nigeria.

Through this project, we applied core web technologies including HTML for structure, CSS for styling, and JavaScript for interactivity. It allowed us to understand how websites are built from concept to completion.

What We Learned

During the development of this project, we gained practical experience in responsive web design and improved our understanding of UI/UX principles.

We learned how to structure web pages properly, write clean code, and apply styling techniques to create visually appealing and functional layouts.

We also improved our problem-solving skills through debugging, testing, and refining different parts of the website.

Cultural Significance

The Eyo Festival is a major cultural event in Yoruba tradition, symbolising respect for ancestors, heritage, and community identity.

This project helps preserve and promote awareness of this cultural festival by presenting it in a modern digital format accessible to a wider audience.
